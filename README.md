# opxy_viz

> Visual-first animation app with bass sequences

## Controls

- **E1** — browse animations (continuous selection)
- **E2** — morph parameter
- **E3** — animation speed
- **K2** — generate new bass sequence
- **K3** — play / stop

## Grid

None required.

## Requirements

- norns
- engine: PolyPerc

## Install

```
;install https://github.com/jamminstein/opxy_viz
```
